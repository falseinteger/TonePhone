# Code of Conduct

## Summary

Be respectful. Be constructive. Focus on the work.

## Expected Behavior

- Treat others with respect
- Give and accept constructive feedback gracefully
- Focus on technical merit
- Assume good intentions

## Unacceptable Behavior

- Personal attacks or insults
- Harassment of any kind
- Trolling or inflammatory comments
- Spam

## Enforcement

Unacceptable behavior may result in:
- Comment removal
- Issue/PR closure
- Temporary or permanent ban

## Scope

This applies to all project spaces: issues, pull requests, and any other project communication channels.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers via email (see SECURITY.md for contact info).

---

This is a small project. Keep it simple. Be an adult.
